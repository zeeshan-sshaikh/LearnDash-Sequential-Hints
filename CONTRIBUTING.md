# Contributing to LearnDash Sequential Hints

First off, thank you for considering contributing to LearnDash Sequential Hints! It's people like you that make the WordPress community such a great place.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct:
- Be respectful and inclusive.
- Focus on what is best for the community.
- Show empathy towards other community members.

## How Can I Contribute?

### Reporting Bugs

- **Check existing issues**: Someone might have already reported the bug.
- **Provide a clear description**: Include steps to reproduce, expected behavior, and actual behavior.
- **Include environment details**: WordPress version, LearnDash version, PHP version, and browser.

### Suggesting Enhancements

- **Open an issue**: Describe the enhancement and why it would be useful.
- **Discuss**: Engage with the community to refine the idea.

### Pull Requests

1. **Fork the repository**.
2. **Create a branch**: Use a descriptive name like `fix/hint-reveal-bug` or `feature/new-counter-style`.
3. **Write code**: Follow WordPress Coding Standards.
4. **Update documentation**: If you're adding a feature, update the README.
5. **Submit the PR**: Provide a clear description of the changes.

## Development Setup

The plugin is designed to be simple to set up:
1. Clone the repo to your `wp-content/plugins` directory.
2. Activate the plugin in WordPress.
3. No build steps are required for the current version.

## Style Guide

- **PHP**: Follow [WordPress PHP Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/php/).
- **JavaScript**: Follow [WordPress JavaScript Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/javascript/).
- **CSS**: Follow [WordPress CSS Coding Standards](https://developer.wordpress.org/coding-standards/wordpress-coding-standards/css/).
